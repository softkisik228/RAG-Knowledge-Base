Документация продуктов TechCorp

1. ERP система "BusinessFlow"
BusinessFlow - это комплексная ERP система для управления бизнес-процессами средних и крупных предприятий.

Основные модули:
- Финансовый учет
- Управление персоналом
- Складское хозяйство
- CRM система
- Отчетность и аналитика

Технические характеристики:
- Веб-интерфейс (React + Node.js)
- База данных: PostgreSQL
- Поддержка микросервисной архитектуры
- API для интеграции с внешними системами
- Мобильное приложение (iOS/Android)

Цены:
- Стартовый пакет (до 50 пользователей): 50,000 руб/месяц
- Корпоративный пакет (до 500 пользователей): 200,000 руб/месяц
- Энтерпрайз (без ограничений): по запросу

2. Платформа "DataInsight"
DataInsight - это платформа для анализа больших данных и машинного обучения.

Возможности:
- Подключение к различным источникам данных
- Визуализация и дашборды
- Предиктивная аналитика
- Автоматизированное построение отчетов
- API для разработчиков

Поддерживаемые форматы данных:
- SQL базы данных (PostgreSQL, MySQL, Oracle)
- NoSQL (MongoDB, Redis)
- Файлы (CSV, JSON, XML, Excel)
- Облачные хранилища (AWS S3, Google Cloud)

3. Мобильное приложение "WorkMate"
WorkMate - корпоративное мобильное приложение для удаленной работы и коммуникации.

Функции:
- Видеоконференции
- Чат и мессенджер
- Управление задачами
- Календарь и планирование
- Файлообменник
- Интеграция с ERP системами

Доступно для платформ: iOS, Android, Web
